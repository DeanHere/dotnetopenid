Project Mission

The goal of this project is to create a C# .NET implementation of OpenID to support both the server, and the client aspects of implementing OpenID.

Phase 1:

  • Port of Grant Morton's Boo .NET libraries
  • Work with Andrew Arnott (or build myself) to update his ASP.NET controls to implement with these libraries

Phase 2:
  • Update/Create a new set of controls to work with Community Server
  • Watch to see how we can help with CardSpace integration
